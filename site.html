Software Used

Resources and Guides

  • HTML Dog: One of the best guides out there! Has tutorials, examples, and references, all for different expertise levels.
  • web.dev: Tutorials in the form of courses! Teaches you a lot of valuable information, including semantic HTML.
  • MDN Web Docs: Another great tutorial I recommend for beginners. It gives you all the basics of web dev!
  • W3Schools: A giant reference guide for many languages, including HTML, CSS, and Javascript.
  • Flexbox Froggy: A cute game that teaches you how to use the CSS Flexbox!
